# Skyvern

## 项目介绍

Skyvern 是面向 Web 的开源 AI 自动化平台。它结合浏览器自动化、视觉模型和 LLM，根据自然语言目标完成网页导航、表单填写、点击、下载和数据抽取。

## 录制方式

- **不要求人工录制。** 用户直接用文字描述任务、网站和希望得到的数据。
- Agent 运行时由浏览器自动获取截图和页面信息，并自行探索操作路径。
- 不需要用户手工截取按钮模板。
- 它不是通过录制视频来学习流程；最终保存的是 Task/Workflow 配置及执行信息。

## 核心能力

| 能力 | 结论 |
| --- | --- |
| AI 语义理解 | 支持 |
| 框架自动截图 | 支持 |
| 人工录制 | 不需要；以文字目标和运行时自动截图为主 |
| 生成回放资产 | 支持 Task/Workflow |
| 一键回放 | 支持通过平台或 API 执行 |
| 自动探索 | 支持，但只限 Web 页面 |
| Windows 桌面 App | 不支持 |

## 优势

- 可以直接描述目标和待提取数据。
- 能探索动态网页，不完全依赖固定 CSS/XPath。
- 适合表单、下载、数据提取和网站结构有变化的场景。
- 社区热度和更新活跃度较高。

## 局限和风险

- 不支持 Wind、同花顺等 Windows 客户端。
- 保存的是 Skyvern Task/Workflow，不一定能导出为独立 Playwright 源码。
- 每次 Agent 执行可能重新规划，确定性弱于固定脚本。

## 不符合要求的核心原因

1. **只支持 Web，不支持桌面 App。**
2. 回放资产主要依赖 Skyvern 的平台和运行时，不是通用独立脚本。
3. AI 自主探索存在执行路径和结果不确定性。

## 适用结论

如果竞品数据全部来自网页，它非常接近“说明想要的数据，Agent 自己探索并抓取”的目标；桌面场景不适用。
